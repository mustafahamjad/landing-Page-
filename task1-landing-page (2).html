Web Basics

When you open a website, your browser (the client) sends a request to a server using HTTP or HTTPS. The server processes that request and sends back the files needed to display the page, such as this one.

Client-Server Model

The browser asks for a page, and a server responds with the content.

HTTP / HTTPS

The protocol used to transfer data between the browser and the server. HTTPS adds encryption.

Domains and Hosting

A domain name points to a server, and hosting is where the website files actually live.

Core Technologies

HTML

Provides the structure and content of a web page.

CSS

Controls how the page looks, including colors, fonts, and layout.

JavaScript

Adds interactivity to a page. This page does not use any JavaScript.

About This Page
